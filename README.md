# AI-Powered Adaptive Study Group Matching Platform

An intelligent platform designed to connect students based on their learning styles, subjects of interest, and collaboration tendencies. The platform uses AI to optimize study group formations and buddy matching for enhanced academic collaboration.

## 🚀 Features

- **AI-Powered Matching**: Intelligent algorithms that match students based on competency levels and learning preferences.
- **Dynamic Study Groups**: Create or join study groups tailored to specific subjects.
- **User Preferences**: Detailed profiles including learning styles (Visual, Auditory, etc.) and communication preferences.
- **Real-time Notifications**: Stay updated on matches and group activities.
- **AI Assistant Widget**: Integrated assistant to help navigate the platform and provide study insights.
- **Responsive Dashboard**: Premium Bento-style grid layout with glassmorphism aesthetics.

## 🛠️ Technology Stack

### Backend
- **Framework**: FastAPI (Python)
- **Database**: SQLAlchemy with SQLite
- **AI Engine**: Custom AI services for adaptive matching
- **Security**: JWT-based authentication

### Frontend
- **Framework**: React.js with Vite
- **Styling**: Tailwind CSS (Premium Glassmorphism Design)
- **Icons**: Lucide React

## 📦 Installation & Setup

### Prerequisites
- Python 3.9+
- Node.js 16+

### Backend Setup
1. Navigate to the `backend` directory.
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the server:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend Setup
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
